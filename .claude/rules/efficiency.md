If you have to update a file or multiple files with a regular/repeated edit try to do it with bash, rather than using a ton of LLM-tool calls. Try to save the user context-tokens where you can. But don't cripple yourself to do it. If you're just making normal edits, use your edit tool as normal. DO NOT USE BASH JUST BECAUSE YOUR EDIT/WRITE TOOL FAILED. This rule is for REPETITIVE EDITS.
